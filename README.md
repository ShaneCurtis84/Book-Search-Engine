# Book-Search-Engine

  ![license](https://img.shields.io/badge/License-MIT-green.svg)

  ## Description

A MERN stack application to be able to search and save your favourite books to your user profile. 
  
  This application was created using 
  - Node JS/Express
  - Mongo DB/ Mongoose
  - React
  - Apollo GraphQL
  - Google books API
